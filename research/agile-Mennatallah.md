# Why Do Companies Use Agile and Git?

## Agile

**Definition:**
Agile is a way of managing projects that helps teams work faster and handle changes more easily.

**Why do companies use Agile?**
- It helps teams finish work faster.
- It improves communication.
- It makes it easier to deal with changes.
- It helps deliver better results.

## Git

**Definition:**
Git is a version control system used to track changes in code and help developers work together.

**Why do companies use Git?**
- It saves different versions of the code.
- It helps developers work as a team.
- It makes it easy to go back to older versions.
- It supports branches and pull requests.
